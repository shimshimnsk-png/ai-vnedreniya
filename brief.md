A: Мы делаем правки до тех пор, пока вы не скажете «ок». Одна итерация правок включена в любой тариф. Мы заинтересованы в результате — наши клиенты приходят по рекомендациям.

**Q: Нужны ли технические знания для работы с системой?**  
A: Нет. Передаём готовое решение и объясняем как им пользоваться. Обычно это занимает 15–20 минут.

**Q: Какие интеграции поддерживаются?**

### Отправка формы (Telegram Bot API)
```ts
const sendToTelegram = async (data: FormData) => {
  const text = `
🆕 Новая заявка с сайта

👤 Ниша: ${data.niche}
🛠 Услуга: ${data.service}
🏢 Проект: ${data.name}
📝 Описание: ${data.description}
📬 Контакт: ${data.contact}
  `.trim();

  await fetch(`https://api.telegram.org/bot${BOT_TOKEN}/sendMessage`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ chat_id: CHAT_ID, text, parse_mode: "HTML" }),
  });
};
```

> `BOT_TOKEN` и `CHAT_ID` — через `.env`: `VITE_BOT_TOKEN`, `VITE_CHAT_ID`

### Убрать из кода
- `https://www.transparenttextures.com/patterns/carbon-fibre.png` (App.tsx:263) — внешняя зависимость. Заменить на SVG-паттерн или убрать.

### Производительность
- Все изображения: WebP, < 150KB
- Анимировать только `transform` и `opacity`
- `will-change: transform` на анимированных blob-элементах
- `whileInView` с `viewport={{ once: true }}` — не повторять анимацию

### Доступность
- Минимальный touch-target: 44×44px
- Все кнопки имеют `aria-label` или понятный текст
- Форма: `autoComplete`, правильные `type` у `<input>`
