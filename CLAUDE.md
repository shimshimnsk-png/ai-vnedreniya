# ai-vnedreniya

Проект: лендинг + интерактивная анкета (мини-апп) для продажи ИИ-услуг.
Бренд: **СамостоятельныеЗапуски** — внедрение ИИ-решений.
Контакт: @makedonskiy (Telegram)

## Стек
- Vite + React + TypeScript
- Tailwind CSS v4 (`@import "tailwindcss"` + `@theme` в CSS)
- Framer Motion (`motion/react`)
- Lucide React (иконки)

## Структура
- `src/App.tsx` — лендинг (переключает вид на MiniApp по кнопке или `?view=mini`)
- `src/MiniApp.tsx` — интерактивная анкета (4 шага: welcome → ниша → услуга → детали → success)
- `src/index.css` — Tailwind v4 + шрифт Inter

## Аудитория лендинга
- Эксперты / Коучи (инфобизнес)
- Блогеры
- Малый бизнес

## Услуги и цены (из MiniApp)
- AI-Ассистент для продаж — от 25к, 3-5 дней
- AI Контент-машина — от 20к, 2-4 дня
- AI База знаний — от 40к, 5-7 дней
- Системная Автоматизация — от 15к, 2-3 дня
- Консультация — бесплатно, 15 мин
